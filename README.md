# MagnOS
A magnet themed operating system. Includes a taskbar with a customizable to-do spot, draggable icons and windows, and a screensaver to keep things moving.

Welcome! Here you can find the code behind MagnOs right now. This operating system was made using the JamOS guide from Hack Club. The background of the OS is the Windows "Ribbons" screensaver!

Disclaimer: AI was used for speeding up VS code editing through suggestions. Also used for debugging the icon dragging to prevent it from opening after being dragged, and to make the pen size picker work in MagSketch.
