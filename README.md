# MagnOS
A magnet themed operating system

Welcome! Here you can find the code behind MagnOs right now. This operating system was made using the JamOS guide from Hack Club. The background of the OS is the Windows "Ribbons" screensaver!

Disclaimer: AI was used for speeding up VS code editing through suggestions, as well as for debugging the icon dragging to prevent it from opening after being dragged.
